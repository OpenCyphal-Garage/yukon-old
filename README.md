# Installation & Development Instructions

``` bash

# install frontend
cd frontend
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

#install backend 
cd ../backend
# install dependencies
pip install todo

# run server
python app.py
```

## Or run setup.sh

(You have to run the backend manually after)
